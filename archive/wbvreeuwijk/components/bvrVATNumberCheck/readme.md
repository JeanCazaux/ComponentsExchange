### Overview
This component checks and validates a VATNumber from the following european countries:

AT-Austria  
BE-Belgium 
BG-Bulgaria 
CY-Cyprus 
CZ-Czech Republic 
DE-Germany 
DK-Denmark 
EE-Estonia 
EL-Greece 
ES-Spain 
FI-Finland 
FR-France 
GB-United Kingdom 
HU-Hungary 
IE-Ireland 
IT-Italy 
LT-Lithuania 
LU-Luxembourg 
LV-Latvia 
MT-Malta 
NL-The Netherlands 
PL-Poland 
PT-Portugal 
RO-Romania 
SE-Sweden 
SI-Slovenia 
SK-Slovakia 

The component takes the VATNumber and the country code either in ISO2 or ISO3 code.
