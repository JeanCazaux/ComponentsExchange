### Overview
In order to use S3 of Amazon, we have made somme components : 
- tS3Connection
- tS3BucketCreate/List/EXist/Detlete
- tS3Object Download/Upload/List/Exist/Delete

Theses components use Amazon Library, present in the zip file.
